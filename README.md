## 玩法说明

点击角色头像可查看角色状态卡片。

角色会自主在工位区、茶水间、洗手间之间移动，也会工作、喝咖啡、发呆、自言自语。

两个角色靠近时有概率触发双人对话，对话内容随机生成，称呼各有不同。

茶水间的咖啡机和自动贩卖机可以被角色使用，有对应动画和音效。

左侧面板可以控制角色的显示与隐藏，右侧面板显示实时事件日志和数据统计。

**v2.0 新增：** 优化自定义角色上传。可以上传透明底 PNG，也可以直接在内置像素画编辑器里画一个（15×15 格子，支持画笔/橡皮擦/吸管工具，提供色卡和色轮，可下载）。
